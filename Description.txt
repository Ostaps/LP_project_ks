Детальний опис проектів і технологій

1) Проект - Epam_Stones_Project

Опис проекту:

Цей проект був створений на основі використання такий frameworks як Spring & Hibernate.
Також в якості бази даних було використано MySQL.В основу проекту також лягли JSP i JSTL
з поєднанням Css/HTML для оформлення вигляду сторінок.UI було створено на основі шаблону, 
оскліки більша увага приділялася саме логіці програми.Загалом проект - це свого роду 
CRUD application.

Можливості:

Користувач має можливість Create/Read/Update/Delete коштовностей(магазин коштовностей).Все
це відбувається динамічно.

IDE and Others:

Eclipse/MySQL Workbench/Tomcat/Brackets(сама програма може відображатися у Eclipse або ж у браузері)

Steps for starting:

Для того, щоб запустити проект потрібно переключитися у Eclipse на Java EE.Також потрібно
налаштувати свій MySQL workbench.Це робиться у конфіг файлі.Просто вказується "користувач" та 
"пароль" від вашого MySQL.Також для запуску application потрібно в Eclipse налаштувати Tomcat Server.
Скачується tomcat і знизу в Eclipse у вкладці Servers налаштовується.Якщо все це зроблено, то можна запускати
application.

2) Проект - InternetMarket

Опис проекту:

Цей проект трохи відрізняється від попереднього, оскільки в ньому використано більший стек технологій.
А саме було додано Spring Security для входу користувача.Тепер можна входити безпосередньо в application 
під різними користувачами.(admin/user).Відповідно для admin можливо здійснювати маніпуляції з товарами(CRUD).
UI також було вибрано шаблоном.Також було застосовано весь стек web-технологій що і в попередньому проекті, але
ще додано JS/Ajax.

Можливості:

Суть даного проекту полягає в реалізації інтернет-магазину.Користувач(user) може заходити(логінуватися)
здійснювати покупки,а admin може маніпулювати продуктами.

IDE and Others:

Intellij Idea/MySQL Workbench/Brackets/Maven/Tomcat Server

Steps for starting:

Для того, щоб запустити цей проект потрібно мати IDE Intellij.Також потрібно імпортувати це проект як Maven,
оскільки в цьому проекті було використано Maven для автоматизації збірки проекту.Аналогічно потрібно налаштувати 
в Intellij Tomcat Server.У .config файлі потрібно налаштувати MYSql.Проект потрібно відкривати з браузера за localhost,
що також можна робити з попереднім проектом.
